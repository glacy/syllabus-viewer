## Tabla de sesiones y resultados de aprendizaje

| Sesión | Título | Resultados de aprendizaje |
|--------|--------|---------------------------|
| 1 | Introducción al frontmatter | <ul><li>Comprender el concepto de frontmatter y su función semántica</li><li>Distinguir entre estructura visual y estructura semántica de un documento académico.</li><li>Identificar las limitaciones de la estructuración puramente visual en la publicación académica</li></ul> |
| 2 | Frontmatter en contextos académicos | <ul><li>Identificar estructuras de frontmatter en diferentes contextos</li><li>Diseñar un frontmatter académico básico que describa correctamente un documento</li><li>Reconocer buenas prácticas y errores comunes en el uso de metadatos académicos.</li></ul> |
| 3 | Diseño semántico de metadatos | <ul><li>Comprender la diferencia entre metadatos sintácticos y metadatos semánticos.</li><li>Diseñar metadatos consistentes, reutilizables y escalables.</li><li>Identificar errores semánticos comunes en documentación académica.</li></ul> |
| 4 | Herencia y reutilización de frontmatter | <ul><li>Entender la herencia de metadatos en sistemas documentales</li><li>Diseñar un frontmatter base reutilizable para cursos académicos</li><li>Definir reglas de sobrescritura controlada de metadatos</li></ul> |
| 5 | Estructura pedagógica con myst.yml | <ul><li>Comprender el rol de `myst.yml` como estructura pedagógica explícita.</li><li>Declarar estructuras curriculares mediante el campo `toc`</li><li>Diseñar progresiones didácticas navegables</li><li>Configurar la identidad visual y experiencia de usuario mediante el campo `site`</li></ul> |
| 6 | Proyecto integrador | <ul><li>Integrar frontmatter, configuración global y estructura de tabla de contenidos en un sistema coherente</li><li>Producir un artefacto académico reproducible y semánticamente correcto</li></ul> |
|  | Banco de ejemplos comentados | <ul><li>Proveer plantillas de metadatos listas para usar.</li><li>Visualizar ejemplos de estructura para diferentes tipos de documentos.</li></ul> |
|  | Jupyter Book vs MyST | <ul><li>Distinguir las diferencias entre Jupyter Book y el ecosistema MyST.</li><li>Comprender la evolución de la configuración de estructura de documentos.</li></ul> |
